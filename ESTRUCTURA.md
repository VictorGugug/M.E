# Estructura actual

## Archivos principales

- `index.html`: pantallas, ventanas, escritorio, apps y Panel de control.
- `styles.css`: estética Windows XP, tema oscuro, boot, WMP, navegador y apps.
- `script.js`: login, boot, ventanas, grilla, sonido, navegador, Paint, calculadora y Windows Media Player local.
- `server.js`: servidor local en `http://127.0.0.1:3000/`.

## Flujo

1. Boot BIOS.
2. Selección manual de arranque.
3. Boot XP con barra azul.
4. Login en español.
5. Escritorio XP.
6. Ventanas, apps y configuración.

## Configuración

- Tema normal u oscuro.
- Boot completo activado/desactivado.
- Animación de escritura BIOS.
- Componentes BIOS visibles: CPU, GPU, RAM, SSD, placa base, refrigeración, fuente, periféricos, monitor y DMI.
- Escritorio con grilla.
- Tamaño de iconos.
- Doble clic o clic simple.
- Sonidos XP.
- Biblioteca local de música desde `NHB/songs.json`.
